不依赖浏览器的web全栈

PhantomJS是一个自带JavaScript API的无头WebKit脚本。 它对各种web标准有快速和原生化的支持，包括DOM处理、CSS选择器、JSON、Canvas和SVG。

下载 v2.1 开始学习 

// 简单的 Javascript 例子

console.log('Loading a web page');
var page = require('webpage').create();
var url = 'http://phantomjs.org/';
page.open(url, function (status) {
  //Page is loaded!
  phantom.exit();
});
